# Tech Stack

**Cursed Energy** — JJK-style hand-tracking cursed techniques (Vite + React + Three.js + MediaPipe)

---

## Core

| Layer | Technology |
|-------|------------|
| Runtime | JavaScript (ES modules) |
| Framework | **React** 18 |
| Language | **TypeScript** 5 |
| Build & dev server | **Vite** 5 |
| Routing | **React Router** 6 |

---

## UI & Styling

| Layer | Technology |
|-------|------------|
| Component library | **shadcn/ui** (Radix UI primitives) |
| Styling | **Tailwind CSS** 3 |
| CSS tooling | PostCSS, Autoprefixer |
| Icons | **Lucide React** |
| Toasts | **Sonner** + Radix Toast |

---

## 3D & Graphics

| Layer | Technology |
|-------|------------|
| 3D engine | **Three.js** |
| Hand tracking | **MediaPipe Hands** (CDN) |

---

## Data & State

| Layer | Technology |
|-------|------------|
| Server state / fetching | **TanStack Query (React Query)** 5 |
| Forms & validation | **React Hook Form** + **Zod** + **@hookform/resolvers** |

---

## Other Libraries

| Purpose | Technology |
|---------|------------|
| Theme (dark/light) | **next-themes** |
| Class names & variants | **clsx**, **tailwind-merge**, **class-variance-authority** |
| Dates | **date-fns** |
| Charts | **Recharts** (available) |

---

## Dev & Tooling

| Purpose | Technology |
|---------|------------|
| Linting | **ESLint** (TypeScript, React, React Hooks, React Refresh) |
| Testing | **Vitest** + **Testing Library** (React, Jest DOM) |
| Dev plugin | **lovable-tagger** (dev only) |

---

## Summary

**Vite · React 18 · TypeScript · Tailwind CSS · shadcn/ui (Radix) · Three.js · MediaPipe Hands · React Router · TanStack Query · React Hook Form · Zod**
