## âš¡ Project Overview

**Cursed Energy** is a creative project that explores iconic Domain
Expansions and cursed techniques from the popular anime *Jujutsu
Kaisen*.\
The main objective of this project is to visually demonstrate how
specific **hand gestures activate different cursed techniques**,
inspired by the anime's power system.

This project focuses on:

-   Visual representation of hand signs\
-   Mapping gestures to domain expansions\
-   Cinematic cursed energy effects\
-   Creative showcase and anime-inspired presentation

It is built as a tribute project and is intended purely for creative and
educational purposes.
